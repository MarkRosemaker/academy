Demonstrativpronomen[Bearbeiten]

Anmerkung:

SingularPlural
etwas ist näher (zeitlich / örtlich) dranthisthese
etwas ist weiter wegthatthose

Mit this beschreibt man etwas, was gerade begonnen hat oder beginnen wird. that wird verwendet, wenn es bereits vorbei ist. Geht es um räumlich Nähe, benutzt man this, ist es etwas oder jemand in der Ferne, so benutzt man that. Die Pluralformen these und those verhalten sich genauso zueinander.

Worttrennung:

this

Aussprache:

IPA: [ðɪs]
Hörbeispiele:  this (Info)  this (US-amerikanisch) (Info)

Bedeutungen:

[1] kennzeichnet ein Ding oder eine Person
[1a] räumlich: in der Nähe
[1b] zeitlich: etwas ist gerade im Gange oder wird beginnen

Beispiele:

[1a] This is a very good table.
Das (hier) ist ein sehr guter Tisch.
[1b] This is a great game.
Das ist ein großartiges Spiel.
[1b] "All this came as something of a relief to his guest." [1]
