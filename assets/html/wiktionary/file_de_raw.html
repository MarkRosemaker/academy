Substantiv[Bearbeiten]

Singular

Plural

the file

Worttrennung:

file, Plural: files

Aussprache:

IPA: [faɪl], Plural: [faɪls]
Hörbeispiele:  file (US-amerikanisch) (Info), Plural:  files (US-amerikanisch) (Info)

Bedeutungen:

[1] Behältnis zur geordneten Aufbewahrung von Briefen oder anderen Dokumenten
[2] Sammlung oder Satz geordneter Informationen oder Schriftstücke über eine Person oder Sache
[3] Informatik: unter einem Namen als Einheit gespeicherte Sammlung von Informationen
[4] mehrere einzeln hintereinander gehende oder stehende Leute

Herkunft:

im frühen 16. Jahrhundert aus dem Französischen, zuerst im Sinn eines Dokumente haltenden Dings belegt, seit dem späten 16. Jahrhundert im Sinn der Leute, seit dem frühen 17. Jahrhundert im Sinn gesammelter und systematisch geordneter Papiere, seit den 1950er Jahren im Sinn der Informatik[1][2]

Synonyme:

[4] single file

Unterbegriffe:

[3] data file

Beispiele:

[1] We collect all orders in a file.
Wir sammeln alle Bestellungen in einem Ordner.
[3] Never delete this file.
Löschen Sie niemals diese Datei.
